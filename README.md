# Microcontroller Engineering Gallery

This repository contains selected embedded systems and microcontroller projects developed during robotics and automation engineering coursework.

The projects focus on low-level hardware control, sensor integration, motor control, and autonomous robotic behavior using AVR-C and Arduino-based systems.

## Featured Projects

### 1. Autonomous Line Follower Robot

Implemented a line-following robot using dual IR sensors and differential motor control.

Features:

* Bang-Bang control mode
* Proportional (P) control mode
* PWM motor speed control
* ADC sensor calibration
* Obstacle and line detection logic

Technologies:

* AVR-C
* PWM motor control
* ADC sensor processing
* Embedded control systems

Project media:

* Demonstration video included

---

### 2. PWM Motor & Sensor Control

Implemented low-level motor control and analog sensor processing for robotics applications.

Topics explored:

* PWM generation
* ADC initialization and calibration
* Sensor threshold processing
* Differential drive logic
